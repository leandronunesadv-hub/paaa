⚖️ Relatórios PAA

Sua planilha é privada — por isso usamos Google OAuth, que autentica diretamente com a sua conta Google sem precisar tornar a planilha pública. Você precisará de um Client ID gratuito. Siga os 5 passos abaixo:

  1. 1
    Acesse o Google Cloud Console Abra console.cloud.google.com com a mesma conta Google da planilha.
  2. 2
    Crie um projeto Clique no seletor de projetos (topo da página) → "Novo projeto" → dê um nome (ex: PAA) → "Criar".
  3. 3
    Habilite a Google Sheets API Menu lateral → "APIs e Serviços" → "Biblioteca" → pesquise Google Sheets API → clique nela → "Ativar".
  4. 4
    Configure a Tela de Consentimento Menu lateral → "APIs e Serviços" → "Tela de consentimento OAuth" → Tipo: Externo → preencha nome do app e e-mail de contato → "Salvar e continuar" em todas as telas.
  5. 5
    Crie o Client ID e configure a origem Menu → "Credenciais" → "+ Criar credenciais" → "ID do cliente OAuth" → Tipo: Aplicativo da Web.

    Em "Origens JavaScript autorizadas", adicione:
    • http://localhost:8080 (se abrir via servidor local)
    • null não funciona mais — se abrir como arquivo local (file://), use um servidor: no terminal, na pasta do arquivo, execute python3 -m http.server 8080 e abra http://localhost:8080

    Clique em "Criar", copie o ID do cliente e cole no campo ao lado.
💡 Dica rápida: Se não quiser instalar nada, hospede este arquivo no GitHub Pages ou Netlify (gratuito) e adicione a URL deles como origem autorizada no Client ID.
